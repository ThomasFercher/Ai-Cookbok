# SmartChef Api
Rest Api using Flask and OpenAi Api.

# Requirements
Python3 

git clone 
cd 
python3 -m venv venv
. venv/bin/activate 


## Dependencies
Flask
requests
pymongo

install with pip install



To Start: flask --app app run 

To Debug: flask --app app --debug run \n
